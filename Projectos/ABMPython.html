Sistema de Ventas

Se a침ade una funcionalidad que permite realizar ventas a un cliente en especifico ("Generar Venta"), pudiendo elegir varias cantidades de productos diferentes, y validando el stock disponible para no romper el programa. Ordenes de Ventas

Cada venta realizada se almacena con un codigo unico para que el usuario pueda consultar su detalle en el momento que desee ("Consulta Pedidos"). En el nuevo submenu se permite cambiar el estado de un pedido simulando la entrega a domicilio.

Listados de Stock

Se permite generar listados de Stock en funcion de lo que usuario necesite en el momento de consultar al sistema ("Listar Productos"). Una vez ingresado, el submenu permite generar listados de productos en stock, o inclusive de los productos vencidos, listando todo en orden creciente por codigo de producto.
